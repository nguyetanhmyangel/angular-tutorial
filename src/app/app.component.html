Pipes example

Trong Angular chia làm hai loại Pipe là pure pipe và impure pipe.

  • Pure pipe: chỉ thực hiện thay đổi khi đầu vào thay đổi. (các immutable objects, primitive type: string, number, boolean, etc): lowercase, uppercase, date, etc.
  • Impure pipe: thực hiện thay đổi mỗi chu kỳ của Change detection (chu kỳ kiểm tra xem các state trong ứng dụng có thay đổi không để update lên view): async, json, etc.

Upper-Lower case

  • {{toString | uppercase}}
  • {{toString | lowercase}}

Date case

  • {{toDate}}
  • {{toDate | date:'dd/MM/y'}}
  • {{toDate | date:'dd/MM/yy HH:mm'}}
  • {{toDate | date:'dd/MM/yy HH:mm':'GMT +7'}}

Percent case

  • {{percentNumber | percent}}
  • {{percentNumber | percent: '4.3-5'}}

Currency case

  • {{price | currency: 'VND'}}
  • {{price | currency:'VND':true:'4.3-5'}}

Decimal case

  • {{price | number}}
  • {{price | number:'.1-4'}}

Slice case

  • {{i}}
  • {{myStr | slice:3:7}}
  • {{myStr | slice:3:-2}}
  • {{myStr | slice:6}}
  • {{myStr | slice:-6}}

Với chuỗi myStr = "abcdefghijk"; chúng ta có start và end index như bên dưới

Json case

for Address Class 
 {{address1 | json}}
for Array of Address
or Person Name Property 
 {{person.pname | json}}
for Person class 
{{person | json}}

Custom case

1. Using welcome Pipe

{{employee.name | welcome}}

2. Using strformat Pipe

{{message | strformat:'+'}}

3. Using division Pipe

Dividend:
Divisor:

Division Result: {{dividend | division: divisor}}

4. Using repeat Pipe

{{employee.name | repeat:10}}

5. Using myjson Pipe

{{employee | myjson}}
{{employee | myjson:0:'id,age'}}
{{employee | myjson:1:'id,name'}}

6. Using myuppercaseone and myuppercasetwo Pipe

{{message | myuppercaseone}}
{{message | myuppercasetwo:'+'}}

7. Chaining of Custom Pipes

{{employee.name | repeat:5 | myuppercaseone}}
